Doubly Linked List

This linear time data structure is comprised of 0-many nodes that store a reference to the node that comes after them as well as the node that comes before them. Basic insert actions use constant time while all other operations use linear time. The trade-off with a doubly linked list is that traversing the list becomes easy but you offer up consuming more memory.

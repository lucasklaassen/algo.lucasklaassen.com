Graph

Toggle Description

This data structure stores nodes which are called verticies that are connected to other nodes using edges. An edge is simply a mapping from one vertex to another. (ie. vertex a is connected to vertex b). Graphs are useful for storing information such as a social network which shows how people are connected through mutual friends. It accomplishes searching in linear time using the depth first search and breadth first search algorithms. Both of these algorithms remember nodes they have already visited allowing them to search linearly.
